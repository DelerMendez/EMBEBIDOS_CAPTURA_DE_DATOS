Detailed Description

Implementación de tareas principales del sistema para lectura de sensor, procesamiento de señales y envío de datos mediante ESP-NOW.

Este archivo define dos tareas principales:

  • Lectura continua del sensor MAX30100 (frecuencia cardíaca y SpO2).
  • Procesamiento de señales y envío periódico de datos promediados.

Utiliza FreeRTOS para la gestión de tareas y colas, permitiendo desacoplar la adquisición de datos del procesamiento y transmisión.

Se implementa una ventana de promediado de datos para mejorar la estabilidad de los valores enviados (BPM, SpO2, señales IR y RED).
