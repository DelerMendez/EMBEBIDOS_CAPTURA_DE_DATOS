Detailed Description

Interfaz del módulo de procesamiento de señal para cálculo de BPM y SpO2.

Este archivo define las estructuras y funciones necesarias para procesar las señales provenientes del sensor MAX30100. Permite calcular la frecuencia cardíaca (BPM), estimar la saturación de oxígeno en sangre (SpO2) y detectar la presencia del dedo en el sensor.

El módulo está diseñado para ser utilizado en sistemas embebidos basados en ESP-IDF, donde las muestras del sensor son procesadas de forma continua.

Note
Las funciones no son completamente thread-safe; se recomienda utilizarlas dentro de una misma tarea o con mecanismos de sincronización.
