Detailed Description

Interfaz del módulo de control del display.

Este archivo define las funciones públicas para la gestión del display, incluyendo inicialización, visualización de estados del sistema y actualización de datos provenientes del sensor IMU.

El módulo garantiza acceso seguro al display mediante mecanismos internos de sincronización (mutex), permitiendo su uso desde múltiples tareas.

Note
La implementación se encuentra en app_display.c.

◆ app_display_init()

esp_err_t app_display_init ( void )

Inicializa el display y la interfaz gráfica.

Configura el driver del display, inicializa la UI y prepara los recursos necesarios para su uso.

Returns
ESP_OK si la inicialización fue exitosa.
Código de error en caso contrario.

Realiza:

  • Creación del mutex
  • Inicialización del driver GC9A01
  • Inicialización de la UI
  • Muestra mensaje inicial
Returns
ESP_OK si la inicialización fue exitosa.
ESP_ERR_NO_MEM si falla la creación del mutex.
ESP_ERR_TIMEOUT si no se puede adquirir el mutex.
