Detailed Description

Interfaz de control para el display LCD basado en el controlador GC9A01 utilizando ESP-IDF.

Este archivo define las funciones necesarias para inicializar, controlar y dibujar en un display GC9A01. Incluye operaciones básicas como dibujo de píxeles, llenado de regiones, control de energía, y manejo de buffers (incluyendo soporte opcional para PSRAM).

Las funciones están diseñadas para ser utilizadas en sistemas embebidos con ESP-IDF. Dependiendo de la implementación en el archivo fuente, algunas funciones pueden ser bloqueantes debido al uso de comunicación SPI y no son inherentemente thread-safe si se accede al bus desde múltiples tareas.

◆ GC9A01_Init()

void GC9A01_Init ( )

Inicializa el display GC9A01.

Configura la interfaz de comunicación (usualmente SPI), inicializa registros internos del controlador y prepara el display para su uso.

Note
Esta función es bloqueante y debe ejecutarse antes de cualquier operación gráfica.
No es thread-safe si múltiples tareas acceden al bus SPI simultáneamente.

Inicializa el display GC9A01.

Realiza la secuencia completa de inicialización del driver:

  1. Asigna el buffer en PSRAM si corresponde (CONFIG_GC9A01_BUFFER_MODE_PSRAM).
  2. Inicializa los GPIOs mediante gc9a01_GPIO_init.
  3. Inicializa el bus SPI mediante gc9a01_SPI_init.
  4. Configura LEDC para PWM de retroiluminación si está habilitado.
  5. Realiza reset por hardware si está habilitado.
  6. Envía la tabla completa de comandos de inicialización lcd_init_cmds.
  7. Configura orientación, modo de color, inversión y salida del modo sleep.
  8. Enciende la pantalla y, en modo buffer, realiza un primer volcado limpio.
  9. Establece la retroiluminación al 100% si el control está habilitado.

Esta función es bloqueante e incluye múltiples retardos requeridos por la hoja de datos del GC9A01. Debe llamarse una única vez desde la tarea de inicialización de la aplicación.

◆ GC9A01_SetBL()

void GC9A01_SetBL ( uint8_t Value)

Ajusta el nivel de brillo de la retroiluminación (Backlight).

Parameters
ValueValor del brillo (dependiente del hardware, típicamente 0–255).
Note
Puede utilizar PWM dependiendo de la implementación.

Ajusta el nivel de brillo de la retroiluminación (Backlight).

Si el modo PWM está habilitado (CONFIG_GC9A01_CONTROL_BACK_LIGHT_MODE), ajusta el ciclo de trabajo del canal LEDC. En caso contrario, conmuta el pin de retroiluminación de forma digital (encendido/apagado). Si el control de retroiluminación no está habilitado en menuconfig, la función no tiene efecto.

Parameters
ValueNivel de brillo deseado, de 0 (apagado) a 100 (máximo brillo). Valores superiores a 100 se recortan a 100.

◆ GC9A01_SleepMode()

void GC9A01_SleepMode ( uint8_t Mode)

Activa o desactiva el modo de suspensión del display.

Parameters
ModeValor que indica el estado:
  • 0: Desactivar modo sleep (display activo)
  • 1: Activar modo sleep (bajo consumo)
Note
Puede implicar retardos internos del hardware (función potencialmente bloqueante).

Activa o desactiva el modo de suspensión del display.

Envía el comando de entrada o salida de modo sleep según el parámetro. Después del comando se espera 500 ms como lo requiere la hoja de datos del GC9A01. Esta función es bloqueante.

Parameters
Mode1 para entrar en modo sleep (pantalla apagada, bajo consumo), 0 para salir del modo sleep (pantalla operativa).
