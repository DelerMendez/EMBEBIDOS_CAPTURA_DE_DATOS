\doxysubsection{Detailed Description}
Tarea de Free\+RTOS para adquisición, procesamiento y gestión de eventos del sensor IMU QMI8658. 

Este módulo implementa una tarea que:\+
\begin{DoxyItemize}
\item Inicializa y calibra el sensor IMU.
\item Lee periódicamente datos de aceleración y giroscopio.
\item Detecta cambios abruptos o eventos significativos.
\item Determina estados de reposo (inactividad prolongada).
\item Envía datos mediante ESP-\/\+NOW y los publica en una cola.
\item Activa alarmas visuales en caso de condiciones específicas.
\end{DoxyItemize}

La lógica incluye:\+
\begin{DoxyItemize}
\item Detección de eventos fuertes (movimientos bruscos).
\item Monitoreo posterior al evento para detectar inmovilidad prolongada.
\item Control de frecuencia de eventos mediante cooldown.
\end{DoxyItemize}

\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item Esta función está diseñada para ejecutarse como una tarea de Free\+RTOS.
\item Utiliza colas (Queue\+Handle\+\_\+t), por lo que es thread-\/safe en ese contexto.
\item Incluye retardos con v\+Task\+Delay, por lo que es una tarea cooperativa.
\end{DoxyItemize}
\end{DoxyNote}

Tarea principal de adquisición y procesamiento del sensor IMU. 

Tarea de Free\+RTOS para adquisición y procesamiento del sensor IMU.

Esta tarea realiza las siguientes funciones en bucle infinito:\+
\begin{DoxyItemize}
\item Lectura periódica del sensor IMU.
\item Cálculo de diferencias entre muestras consecutivas.
\item Detección de eventos abruptos o cambios significativos.
\item Publicación de datos en una cola Free\+RTOS.
\item Envío de datos mediante ESP-\/\+NOW.
\item Monitoreo de estado de reposo tras eventos fuertes.
\item Activación de alarmas si se detecta inactividad prolongada.
\end{DoxyItemize}

\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item Función bloqueante (bucle infinito con v\+Task\+Delay).
\item Diseñada para ejecutarse como tarea de Free\+RTOS.
\item Usa x\+Queue\+Overwrite, por lo que la cola debe ser de longitud 1.
\end{DoxyItemize}
\end{DoxyNote}
