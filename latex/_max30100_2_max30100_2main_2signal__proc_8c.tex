\doxysubsection{Detailed Description}
Procesamiento de señal para cálculo de BPM y estimación de Sp\+O2. 

Este archivo implementa el procesamiento de señales provenientes del sensor MAX30100, incluyendo:\+
\begin{DoxyItemize}
\item Detección de presencia de dedo
\item Cálculo de frecuencia cardíaca (BPM)
\item Estimación de saturación de oxígeno (Sp\+O2)
\end{DoxyItemize}

Utiliza buffers circulares para almacenar muestras y aplicar cálculos estadísticos como promedio y RMS para separar componentes AC/\+\+DC.

\begin{DoxyNote}{Note}
Varias funciones son bloqueantes a nivel computacional (procesamiento). 

No es thread-\/safe; se recomienda acceso desde una única tarea o con protección.
\end{DoxyNote}
