\doxysubsection{Detailed Description}
Configuración global del sistema para la aplicación basada en ESP-\/\+IDF. 

Este archivo define constantes de configuración utilizadas en todo el sistema, incluyendo:\+
\begin{DoxyItemize}
\item Configuración del bus I2C
\item Parámetros de calibración del sensor
\item Configuración de tareas Free\+RTOS
\item Colores para la interfaz gráfica
\item Parámetros de consumo energético (CPU)
\item Umbrales y tiempos para detección de eventos y comunicación ESP-\/\+NOW
\end{DoxyItemize}

Estas definiciones no contienen lógica ejecutable, pero afectan directamente el comportamiento del sistema en tiempo de ejecución.

\begin{DoxyNote}{Note}
Este archivo debe ser incluido por los módulos que requieran acceso a configuraciones globales.
\end{DoxyNote}
