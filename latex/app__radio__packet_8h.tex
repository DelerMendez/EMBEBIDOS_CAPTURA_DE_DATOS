\doxysubsection{Detailed Description}
Definición de la estructura de paquete utilizada para transmisión vía ESP-\/\+NOW. 

Este archivo define el formato de los datos enviados entre nodos mediante el protocolo ESP-\/\+NOW. El paquete contiene información básica de identificación, control de secuencia, marca de tiempo y un campo de texto para datos del sensor u otros mensajes.

La estructura está empaquetada para asegurar que no haya padding entre sus campos, garantizando así una transmisión eficiente y consistente entre dispositivos.

\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item El tamaño del paquete debe mantenerse dentro de los límites de ESP-\/\+NOW (\texorpdfstring{$\sim$}{\string~}250 bytes).
\item El uso de {\bfseries{attribute}}((packed)) evita rellenos de memoria.
\end{DoxyItemize}
\end{DoxyNote}
