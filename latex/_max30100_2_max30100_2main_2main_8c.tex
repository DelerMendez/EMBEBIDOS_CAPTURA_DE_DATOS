\doxysubsection{Detailed Description}
Punto de entrada principal del sistema embebido basado en ESP32. 

Este archivo contiene la función principal {\ttfamily \doxylink{_max30100_2_max30100_2main_2main_8c_a630544a7f0a2cc40d8a7fefab7e2fe70}{app\+\_\+main}}, la cual se encarga de inicializar el sistema, configurar el sensor MAX30100 y arrancar las tareas de la aplicación.

El flujo principal incluye:\+
\begin{DoxyItemize}
\item Inicialización del sensor mediante I2C
\item Configuración por defecto del MAX30100
\item Inicio de tareas del sistema (posiblemente bajo Free\+RTOS)
\end{DoxyItemize}

Este módulo actúa como orquestador del arranque del sistema.

\begin{DoxyNote}{Note}
La función {\ttfamily \doxylink{_max30100_2_max30100_2main_2main_8c_a630544a7f0a2cc40d8a7fefab7e2fe70}{app\+\_\+main}} es el equivalente a {\ttfamily main()} en ESP-\/\+IDF y se ejecuta como una tarea de Free\+RTOS.
\end{DoxyNote}

Función principal de la aplicación. 

Esta función es el punto de entrada del sistema en ESP-\/\+IDF. Se encarga de:\+
\begin{DoxyItemize}
\item Inicializar el sensor MAX30100 mediante configuración I2C
\item Aplicar una configuración por defecto al sensor
\item Iniciar las tareas principales del sistema
\end{DoxyItemize}

En caso de error en cualquiera de las etapas, se registra el fallo y se detiene la ejecución.

\begin{DoxyNote}{Note}
Esta función se ejecuta como una tarea de Free\+RTOS. 

Puede considerarse bloqueante durante la inicialización.
\end{DoxyNote}
