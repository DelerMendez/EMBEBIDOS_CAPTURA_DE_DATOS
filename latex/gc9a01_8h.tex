\doxysubsection{Detailed Description}
Interfaz de control para el display LCD basado en el controlador GC9\+A01 utilizando ESP-\/\+IDF. 

Este archivo define las funciones necesarias para inicializar, controlar y dibujar en un display GC9\+A01. Incluye operaciones básicas como dibujo de píxeles, llenado de regiones, control de energía, y manejo de buffers (incluyendo soporte opcional para PSRAM).

Las funciones están diseñadas para ser utilizadas en sistemas embebidos con ESP-\/\+IDF. Dependiendo de la implementación en el archivo fuente, algunas funciones pueden ser bloqueantes debido al uso de comunicación SPI y no son inherentemente thread-\/safe si se accede al bus desde múltiples tareas.

Inicializa el display GC9\+A01. 

Configura la interfaz de comunicación (usualmente SPI), inicializa registros internos del controlador y prepara el display para su uso.

\begin{DoxyNote}{Note}
Esta función es bloqueante y debe ejecutarse antes de cualquier operación gráfica. 

No es thread-\/safe si múltiples tareas acceden al bus SPI simultáneamente.
\end{DoxyNote}
Inicializa el display GC9\+A01.

Realiza la secuencia completa de inicialización del driver:\+
\begin{DoxyEnumerate}
\item Asigna el buffer en PSRAM si corresponde ({\ttfamily CONFIG\+\_\+\+GC9\+A01\+\_\+\+BUFFER\+\_\+\+MODE\+\_\+\+PSRAM}).
\item Inicializa los GPIOs mediante {\ttfamily gc9a01\+\_\+\+GPIO\+\_\+init}.
\item Inicializa el bus SPI mediante {\ttfamily gc9a01\+\_\+\+SPI\+\_\+init}.
\item Configura LEDC para PWM de retroiluminación si está habilitado.
\item Realiza reset por hardware si está habilitado.
\item Envía la tabla completa de comandos de inicialización {\ttfamily lcd\+\_\+init\+\_\+cmds}.
\item Configura orientación, modo de color, inversión y salida del modo sleep.
\item Enciende la pantalla y, en modo buffer, realiza un primer volcado limpio.
\item Establece la retroiluminación al 100\% si el control está habilitado.
\end{DoxyEnumerate}

Esta función es bloqueante e incluye múltiples retardos requeridos por la hoja de datos del GC9\+A01.
