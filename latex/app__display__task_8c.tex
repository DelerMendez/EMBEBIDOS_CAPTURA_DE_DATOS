\doxysubsection{Detailed Description}
Tarea encargada de la actualización del display con datos del IMU. 

Este módulo implementa una tarea de Free\+RTOS que recibe datos del sensor IMU a través de una cola y los muestra en pantalla.

Flujo de funcionamiento:\+
\begin{DoxyItemize}
\item Inicializa el display en modo de arranque
\item Espera datos provenientes de la cola
\item Actualiza la interfaz gráfica con los nuevos datos del IMU
\end{DoxyItemize}

La tarea se ejecuta de forma continua en un bucle infinito.

\begin{DoxyNote}{Note}
La cola debe ser creada previamente y pasada como parámetro.
\end{DoxyNote}
