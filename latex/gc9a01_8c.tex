\doxysubsection{Detailed Description}
Controlador de pantalla TFT circular GC9\+A01 para ESP-\/\+IDF. 

Este archivo implementa el driver de bajo nivel para la pantalla LCD circular basada en el controlador GC9\+A01, utilizando el bus SPI del framework ESP-\/\+IDF. Soporta modo directo y modo buffer (con opción de PSRAM), control de retroiluminación por GPIO o PWM (LEDC), rotación de pantalla, inversión de color y operaciones de dibujo de píxeles y rectángulos.
