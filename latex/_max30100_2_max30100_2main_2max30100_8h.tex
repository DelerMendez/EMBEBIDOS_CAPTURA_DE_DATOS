\doxysubsection{Detailed Description}
Interfaz del driver para el sensor MAX30100 usando I2C en ESP-\/\+IDF. 

Este archivo define las estructuras de configuración, tipos de datos y funciones necesarias para interactuar con el sensor MAX30100, el cual permite medir frecuencia cardíaca y niveles de oxígeno en sangre (Sp\+O2).

Proporciona una API para:\+
\begin{DoxyItemize}
\item Inicialización del sensor
\item Configuración por defecto
\item Lectura de datos desde el FIFO
\item Verificación del estado del sensor
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Las funciones pueden ser bloqueantes debido al uso del bus I2C y retardos internos. 

No todas las funciones son thread-\/safe; se recomienda sincronización externa si se accede desde múltiples tareas.
\end{DoxyNote}
