\doxysubsection{Detailed Description}
Implementación de tareas principales del sistema para lectura de sensor, procesamiento de señales y envío de datos mediante ESP-\/\+NOW. 

Este archivo define dos tareas principales:\+
\begin{DoxyItemize}
\item Lectura continua del sensor MAX30100 (frecuencia cardíaca y Sp\+O2).
\item Procesamiento de señales y envío periódico de datos promediados.
\end{DoxyItemize}

Utiliza Free\+RTOS para la gestión de tareas y colas, permitiendo desacoplar la adquisición de datos del procesamiento y transmisión.

Se implementa una ventana de promediado de datos para mejorar la estabilidad de los valores enviados (BPM, Sp\+O2, señales IR y RED).
