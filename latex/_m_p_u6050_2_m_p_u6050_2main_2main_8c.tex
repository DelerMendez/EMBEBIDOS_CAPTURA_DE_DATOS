\doxysubsection{Detailed Description}
Aplicación principal para adquisición de datos del MPU6050 y envío vía ESP-\/\+NOW. 

Este archivo implementa un nodo emisor basado en ESP32 que:\+
\begin{DoxyItemize}
\item Inicializa el sensor MPU6050 mediante I2C.
\item Lee periódicamente datos de aceleración, giroscopio y temperatura.
\item Empaqueta los datos en una estructura de log.
\item Envía la información a otro dispositivo mediante ESP-\/\+NOW.
\end{DoxyItemize}

El sistema está diseñado para ejecutarse sobre Free\+RTOS, utilizando un bucle principal bloqueante con retardos periódicos.
