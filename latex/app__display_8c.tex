\doxysubsection{Detailed Description}
Implementación del módulo de control del display. 

Este módulo proporciona una interfaz segura para el acceso al display, utilizando un mutex de Free\+RTOS para evitar condiciones de carrera cuando múltiples tareas intentan actualizar la pantalla.

Funcionalidades principales:\+
\begin{DoxyItemize}
\item Inicialización del display (driver + UI)
\item Manejo de estados visuales (INIT, CAL, RUN, ERROR)
\item Actualización de datos del IMU en pantalla
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Todas las operaciones sobre el display están protegidas con mutex.
\end{DoxyNote}
