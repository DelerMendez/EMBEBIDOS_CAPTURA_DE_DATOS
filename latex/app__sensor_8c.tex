\doxysubsection{Detailed Description}
Módulo de inicialización, calibración y lectura del sensor IMU QMI8658 mediante interfaz I2C. 

Este archivo implementa las funciones necesarias para:\+
\begin{DoxyItemize}
\item Configurar el bus I2C en modo maestro.
\item Inicializar el sensor QMI8658.
\item Calibrar el giroscopio.
\item Leer datos del sensor IMU.
\end{DoxyItemize}

El módulo abstrae la configuración de bajo nivel del bus I2C y delega la interacción específica del sensor al driver qmi8658.

\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item Las funciones dependen del driver I2C de ESP-\/\+IDF.
\item No es thread-\/safe si múltiples tareas acceden simultáneamente sin sincronización.
\item Algunas funciones pueden ser bloqueantes dependiendo del driver subyacente.
\end{DoxyItemize}
\end{DoxyNote}
